Classes

a class is the building block that leads to Object-Oriented Programming. It is a user-defined data type, that holds its own data members and member functions, which can be accessed and used by creating an instance of that class. It is the blueprint of any object. Once we have written a class and defined it, we can use it to create as many objects based on that class as we want.

Objects

Object is an instance of a class. All data members and member functions of the class can be accessed with the help of objects. When a class is defined, no memory is allocated, but memory is allocated when it is instantiated (i.e. an object is created).

Difference Between Class And Object:

Class Object
Class is used as a template for declaring and creating the objects. An object is an instance of a class.
When a class is created, no memory is allocated. Objects are allocated memory space whenever they are created.
The class has to be declared first and only once. The class has to be declared first and only once.
A class can not be manipulated as they are not available in the memory. Objects can be manipulated.
Class does not contain any values which can be associated with the field. Each object has its own values, which are associated with it.
A class is used to bind data as well as methods together as a single unit. Objects are like a variable of the class..
